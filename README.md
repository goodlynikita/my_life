## Важное исправление в этой версии

Раньше после любого изменения было 2.5 секунды задержки перед отправкой
данных в общую базу. Если в этот момент закрыть вкладку или свернуть
браузер на телефоне — изменения не успевали уйти в облако и оставались
только на этом устройстве, из-за чего на других устройствах их не было
видно. Теперь сохранение происходит мгновенно при каждом изменении, плюс
добавлена дополнительная защита: даже если страница закрывается прямо в
момент отправки, браузер всё равно постарается её завершить.

## Что нужно сделать

Загрузи все файлы из этой папки в свой репозиторий на GitHub: **Add file →
Upload files** → выбери всё содержимое (файлы и папки `css`, `js`, плюс
`data.json`) → **Commit changes**.

Если на каком-то устройстве после обновления видно старое — один раз
очисти данные сайта в браузере (в Safari на Mac: Safari → Настройки →
Конфиденциальность → Управление данными веб-сайтов → найти сайт → Удалить),
дальше всё должно работать само, без этого шага.

## Если хочешь сменить пароль

Открой `generate-hash.html`, введи новый пароль, скопируй хэш, вставь в
`js/config.js` вместо `ownerHash` или `coachHash`, сохрани файл обратно в
репозиторий.
