## Важное исправление в этой версии

Раньше при быстром вводе нескольких упражнений подряд два запроса на
сохранение могли улететь в облако почти одновременно, и если более ранний
запрос почему-то долетал позже более позднего — он мог затереть свежие
данные старыми. Из-за этого иногда пропадали только что введённые
упражнения и веса, хотя сама тренировка (тип/группа) сохранялась. Теперь
все сохранения идут строго по очереди, одно за другим — финальный
результат всегда самые последние данные.

## Что нужно сделать

Загрузи все файлы из этой папки в свой репозиторий на GitHub: **Add file →
Upload files** → выбери всё содержимое (файлы и папки `css`, `js`, плюс
`data.json`) → **Commit changes**.

Если на каком-то устройстве после обновления видно старое — один раз
очисти данные сайта в браузере (Safari → Настройки → Конфиденциальность →
Управление данными веб-сайтов → найти сайт → Удалить).

## Если хочешь сменить пароль

Открой `generate-hash.html`, введи новый пароль, скопируй хэш, вставь в
`js/config.js` вместо `ownerHash` или `coachHash`, сохрани файл обратно в
репозиторий.
